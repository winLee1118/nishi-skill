# B 站人物风格样本来源目录

本目录用于整理“倪师数字人”的人物风格蒸馏素材来源。当前只保存来源索引、采样计划和后续样本落盘位置，不保存未授权的完整 B 站视频文件。

## 目录说明

- `source-catalog.json`：按人纪、天纪、地纪、访谈/问答整理的候选视频来源。
- `sampling-plan.md`：后续提取小样本时的采样流程、覆盖比例和质量标准。
- `samples/`：后续只放短片段转写、短截图 OCR、人工摘录的风格样本；不要放完整视频。
- `raw-videos/`：本地下载的视频素材目录，已加入 `.gitignore`，不进入开源仓库。
- `private/`：本地 cookie、下载脚本、运行日志等私有辅助文件，已加入 `.gitignore`。

## 字幕提取

当前本地视频没有软字幕轨，也没有旁挂 `.srt/.ass/.vtt` 文件。可行流程是：

1. 在剪映专业版中导入视频。
2. 使用“智能字幕/识别字幕”生成字幕轨。
3. 使用 `tools/jianying_template_to_srt.py` 从剪映草稿 `template.json` 导出 `.srt`。

示例：

```powershell
python -B tools\jianying_template_to_srt.py "D:\Program Files\JianyingPro Drafts"
```

导出的字幕默认放到 `materials/bilibili-style-sources/private/jianying-subtitles/`。

## 合规边界

- B 站页面链接可以作为来源索引保存。
- 未明确授权的完整视频不进入仓库，也不作为开源资产发布。
- `raw-videos/` 和 `private/` 只用于本机研究流程，不作为项目交付物。
- 如果后续需要本地化处理，应优先使用公开视频自带字幕、UP 主明确授权素材、你本人拥有权利的文件，或只截取极短片段用于风格分析。
- 人物风格蒸馏只学习表达结构、讲课节奏、比喻习惯、追问方式和人格维度；不做声音克隆，不宣称本人在线。

## 当前覆盖

- 人纪：针灸、黄帝内经、伤寒论、金匮要略、临床案例。
- 天纪：易经、天纪总集、紫微斗数、四柱命卦。
- 地纪：阳宅、风水、堪舆。
- 访谈：梁冬对话、长谈式表达。
