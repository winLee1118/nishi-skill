# 人物风格采样计划

## 目标

扩大人物风格蒸馏样本，不追求完整复刻，而是提取稳定可控的表达指纹：

- 开场方式：如何先框定问题、先问条件、先讲结论。
- 推理节奏：从现象、象义、病机、方义或案例推进的顺序。
- 语言颗粒度：短句、重复强调、课堂口吻、反问句、转折句。
- 比喻习惯：生活物件、身体感受、临床场景、家庭空间、卦象图像。
- 个性维度：笃定、直接、带一点课堂幽默，但不过度表演。
- 边界表达：能讲知识和辨证框架，不替用户做个人诊断、处方和剂量决定。

## 第一轮采样配比

建议先做 80 到 120 条短样本，每条 20 到 80 字，覆盖如下：

- 人纪讲课：30 条，重点采“先辨证、再讲方义、再讲禁忌”的表达。
- 人纪案例/问答：20 条，重点采追问方式、病人描述转成辨证要点的方式。
- 天纪易经/紫微：20 条，重点采“先看象、再看动、不要死断”的口吻。
- 地纪阳宅/堪舆：10 条，重点采空间方位、家庭场景、类象解释。
- 访谈长谈：20 条，重点采世界观、人生观、教育式语气和轻松转折。

## 单条样本格式

后续样本建议落到 `samples/*.jsonl`，每行一条：

```json
{"source_id":"renji-acupuncture-001","scene":"人纪/针灸","sample_type":"style","text":"这里只放极短摘录或人工转写摘要","features":["先问条件","课堂口吻","类比"],"notes":"不要收录长段原文"}
```

## 采样流程

1. 从 `source-catalog.json` 选一个来源页面。
2. 优先看是否有公开视频字幕；没有字幕时只做短截图 OCR 或人工短摘。
3. 每个来源先采 3 到 8 条，避免单一视频风格过拟合。
4. 每条样本标注场景、表达功能和可复用风格特征。
5. 把样本汇总后再更新 `knowledge/persona/style-fingerprint-v*.json`。

## 质量标准

- 不采长篇逐字稿。
- 不采明显二创旁白、剪辑者标题党语句。
- 不采非倪师本人表达作为人物风格。
- 同一个结论如果多个视频重复，只保留最具代表性的表达结构。
- 医学内容只作为知识解释和辨证框架，不生成个人治疗指令。
